Add social share buttons on Habakiri theme.

== Description ==

This plugin add social share buttons on Habakiri theme. You can select share button layout balloon or horizontal.

= Support buttons =

* Facebook
* Twitter
* Hatena Bookmark
* Pocket
* Feedly

= GitHub =

https://github.com/inc2734/habakiri-share-buttons/

== Installation ==

1. Upload `Habakiri Share Buttons` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You can setting in Settings > Share Buttons Settings
